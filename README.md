# Predictive Maintenance System — QNX RTOS Concepts

![Platform](https://img.shields.io/badge/Platform-QNX%20Neutrino-blue)
![Language](https://img.shields.io/badge/Language-C-lightgrey)

A Predictive Maintenance System (PMS) predicts failures by analyzing real-time data. Using QNX Neutrino RTOS, it simulates sensor inputs like temperature and vibration, processes data, and detects anomalies. It uses IPC (message passing), multithreading, and real-time scheduling for reliable performance.
